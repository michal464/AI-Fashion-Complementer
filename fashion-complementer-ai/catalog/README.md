# Catalog Images Guide

Place product images in each category folder.
Naming convention: `{style}_{description}_{number}.jpg`

## catalog/images/shoes/
Footwear items to recommend as complements.
Examples:
- casual_white_sneakers_01.jpg
- casual_white_sneakers_02.jpg
- formal_black_oxford_01.jpg
- formal_brown_derby_01.jpg
- sporty_running_shoes_01.jpg
- sporty_running_shoes_02.jpg
- boho_sandals_01.jpg

## catalog/images/bags/
Bags and purses.
Examples:
- casual_canvas_tote_01.jpg
- formal_leather_handbag_01.jpg
- sporty_backpack_01.jpg
- boho_fringed_bag_01.jpg

## catalog/images/watches/
Watches and smart watches.
Examples:
- casual_digital_watch_01.jpg
- formal_silver_watch_01.jpg
- sporty_smartwatch_01.jpg

## catalog/images/shirts/
Tops, t-shirts, blouses (upload these to train the classifier, NOT as recommendations).
Examples:
- casual_white_tshirt_01.jpg
- formal_button_shirt_01.jpg

## catalog/images/pants/
Pants, jeans, trousers.
Examples:
- casual_blue_jeans_01.jpg
- formal_black_trousers_01.jpg
- sporty_joggers_01.jpg

## catalog/images/jackets/
Jackets, coats, blazers.
Examples:
- casual_denim_jacket_01.jpg
- formal_blazer_black_01.jpg
- sporty_track_jacket_01.jpg
- boho_kimono_01.jpg

## catalog/images/accessories/
Belts, scarves, hats, jewelry.
Examples:
- casual_belt_brown_01.jpg
- formal_silk_tie_01.jpg
- boho_necklace_01.jpg
- sporty_cap_01.jpg

## Tips:
- Minimum 5 images per category for good recommendations
- Use clean product photos (white/plain background preferred)
- JPG or PNG format
- Any resolution (will be resized automatically to 224x224)
